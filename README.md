## Pre-requisistos para API

### Crear env

python -m venv venv

### Activar env

venv/Scripts/Activate.ps1

### Instalar dependencias

pip install -r .\requirements.txt

## Correr el proyecto

uvicorn main:app --reload --port 8001

## Pre-requisistos para Front

### Instalar librerias

npm i

## Correr el proyecto

npm run serve
